## Getting Started

### Prerequisites

* Oculus Quest 2
* Unser Haus application

### Installation

1. app. Download [Here](https://drive.google.com/file/d/1iJ5BJFcEqVkFl2a9BUCNxH6RePTwv0KB/view?usp=sharing)
2. Install on your Oculus Quest through Oculus Link
3. Play
